Simple attendance tracking web app built with PHP, JavaScript and MySQL.
It includes user authentication (login/register), session handling, and basic attendance management.
Features:
1. User registration & login
2. Attendance recording
3. Organized panels for users & admin
4. CSRF protected forms

Tech Stack:
1. PHP
2. JS
3. MYSQL
4. HTML & CSS

Setup:
1. Clone the repo
2. Run php -S 127.0.0.1:8000 in your preferred terminal
3. Go to http://127.0.0.1:8000/login/
4. Ask me for Credentials
